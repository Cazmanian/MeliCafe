# MeliCafe
A website created for Eleni's coffee trucked named Meli Cafe!

Stakeholder needs: 
She told me that she wants it to all be on one page and provided me with the colors and typography she wanted used for her website. She wants the website to provide these sections in this hierarchy: about, menu, catering, gallery, location, and reviews. She wants an order online call-to-action button on the hero image on top of the page. She would also like a promotion pop up when you first enter the website.

Created a low-fidelity wireframe for the desktop and mobile version of the website. 

Plan: 
Create a one-page website with stakeholder needs. Use anchor links for a navbar to quickly go to a certain section of the page. Create a pop-up screen when you first arrive to website.
